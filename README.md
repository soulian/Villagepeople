# 정자일로 55 빌리지피플

아파트별 주민 커뮤니티. 상단 드롭다운으로 다른 아파트를 선택해 둘러볼 수 있습니다. **하이텔** 시절 PC통신 터미널 느낌의 디자인을 사용합니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속하세요.

## 메뉴 구조

- **기본메뉴**: 공지사항, 빌리지피플?(서비스 소개)
- **커뮤니티**: 자유게시판, {아파트이름}소식, 개선건의사항

## 기능

- **공지사항**: 운영자 비밀번호 입력 후에만 글쓰기 가능 (기본 비밀번호: `village2024`, 변경 시 `src/data/mock.js`의 `OPERATOR_PASSWORD` 수정)
- **그 외 게시판**: 닉네임 입력 시 해당 이름으로 글·댓글 등록 (미입력 시 익명)
- **아파트 전환**: 상단 헤더의 아파트 이름 옆 ▼ 클릭 → 다른 아파트 선택

데이터는 브라우저 **localStorage**·**sessionStorage**에 저장됩니다.
